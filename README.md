# VisioMusic 
An AI-powered music player that detects your mood using facial recognition.

## Features
* Detects emotions (Happy, Sad, Neutral) using the Camera.
* Plays a matching playlist automatically.
* Built with Python, Flask, and OpenCV.

## How to Run
1. Install requirements: `pip install -r requirements.txt`
2. Run the app: `py app.py`
